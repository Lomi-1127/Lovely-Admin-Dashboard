##关于

Lovely-Admin-Dashboard是一个干净的，视觉上令人愉悦的管理面板模板。  
它摆脱了杂乱无聊的后端，带来柔和的色彩、圆角和流畅的交互。

适合：
-SaaS控制面板
-CMS管理面板
-内部工具
-个人项目
